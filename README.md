# Praktikum PAM M4 - My Profile App
## Deskripsi 
Aplikasi ini dibuat menggunakan Jetpack Compose untuk menampilkan halaman profil sederhana.
## Fitur 
- Menampilkan foto profil
- Menampilkan nama dan bio
- Informasi email
- Nomor telepon
- Lokasi 
- Tombol Edit Profile

## Komponen Compose yang Digunakan
- Column 
- Row 
- Box
- Card
- Text 
- Button 
- Image
